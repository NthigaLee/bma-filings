KEY DESIGN ELEMENTS:

1. MAROON & WHITE THEME
   ├─ Primary: Maroon (#8b1538) - headers, buttons, borders
   ├─ Secondary: Light Gray (#f5f5f5) - page background
   ├─ Tertiary: White (#ffffff) - card backgrounds
   └─ Text: Dark Gray (#333333) - readability

2. INTERACTIVE FEATURES
   ├─ Year dropdown (2023 vs 2024)
   ├─ Company selection checkboxes (all 10 companies)
   ├─ Select All / Clear All buttons
   ├─ Metric selector for trend chart
   └─ Real-time chart updates

3. DATA VISUALIZATION
   ├─ 8 bar charts (2-column layout)
   ├─ 1 line chart (year-over-year trend, full width)
   ├─ 1 comparison table (full width)
   └─ 4 metric cards (responsive grid)

4. RESPONSIVE LAYOUT
   ├─ Desktop (>1200px): Full 2-column chart layout
   ├─ Tablet (768-1200px): Adjusted spacing
   └─ Mobile (<768px): Single column stack

5. CHART TYPES
   ├─ Bar charts: Compare companies across metrics
   ├─ Line chart: Trend comparison between years
   └─ Table: Detailed numeric comparison
